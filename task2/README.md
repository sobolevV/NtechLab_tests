<b>Файл pocess.py<b>

Параметры запуска:
    --folder обязательный параметр, указывающий на папку с изображениями
    -m, --model указывает на тип модели. Допустимые значения (simple, vgg)

Пример: python process.py --folder="test_data" -m="vgg"
